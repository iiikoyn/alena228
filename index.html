Как установить приложение:

  1. Скачайте APK-файл по ссылке
  2. Перейдите в Настройки > Безопасность и разрешите установку из неизвестных источников
  3. Откройте скачанный APK-файл и нажмите "Установить"
  4. После установки откройте приложение "Шуруповёрты"

Требования: Android версии 8.0 и выше

Скачать для Windows

Как открыть файл:

  1. Скачайте архив с программой по ссылке выше
  2. Распакуйте архив с помощью WinRAR или 7-Zip
  3. Откройте папку с распакованными файлами
  4. Запустите файл main.exe
  5. Если появится предупреждение безопасности, нажмите "Дополнительно" и выберите "Выполнить в любом случае"

Требования: Windows 10/11, .NET Framework 4.7.2 или новее
